### Tecnologias Utilizadas
- Linguagem de programação: C#
- Banco de dados: MySQL
- Frameworks: ASP.NET
- IDEs: Visual Studio
- Serviços web: ISBNdb
- Ferramentas de controle de versão: Git
- Bibliotecas: ADO.NET
